## 🔬 Projetos em Destaque

### 🩺 [Bootcamp de Machine Learning](https://github.com/anysabele/Projeto-Bootcamp-ML-Avanti-2026.1)
Projeto Final em equipe do Bootcamp de Machine Learning da Atlântico Avanti, focado na classificação de lixo reciclável. Os passos seguintes foram:

1. Verificação da integridade dos arquivos, consistência dos metadados, qualidade das imagens, distribuição das classes e duplicatas.
2. Separação dos dados para o treinamento do modelo.
3. Treinamento do modelo: ResNet-50.
4. Verificação de resultados.

### 🩺 [Predição de Câncer Cervical com ML & MCDM](https://github.com/elianramon/TCC_UFPA_Eng_Comp)
Meu TCC focado na saúde pública do Norte do Brasil. Utilizei técnicas avançadas de reamostragem e métodos de tomada de decisão multicritério (MCDM) para selecionar o melhor modelo preditivo clássico, alcançando alta eficácia com Random Forest.

### 🌳 [SIOFA - Inteligência Operacional para Fiscalização Ambiental](https://github.com/elianramon/SIOFA_I2A2_final_project)
Desenvolvimento de uma arquitetura que utiliza **Random Forest** para predição de risco ambiental e **Algoritmos Genéticos** para otimização logística na Amazônia Legal. O sistema funciona como um cérebro operacional para converter dados do DETER/PRODES em rotas estratégicas.
